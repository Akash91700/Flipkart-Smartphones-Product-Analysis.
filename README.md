# Flipkart-Smartphones-Product-Analysis.
This project is a data analysis case study based on Flipkart, one of India’s leading e-commerce companies. The dataset includes details such as product category, price segment, ratings, outlet size, outlet location, and sales performance.

The goal of this project is to perform data cleaning, exploratory data analysis (EDA), and visualization to identify key trends that influence customer behaviour and product performance on Flipkart.

Using Python libraries like Pandas, NumPy, Matplotlib, and Seaborn, the project covers:

Handling missing values & duplicates

Formatting and preprocessing data

Visualizing sales by outlet size, location, price segment, and more

Understanding relationships between product ratings, pricing, and sales


The insights generated can help Flipkart optimize seller strategies, improve product placement, and enhance customer experience.
